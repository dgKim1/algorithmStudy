**자바스크립트 map()\[배열메소드]**



&nbsp;.map(group => group.split(",").map(Number))

각 "1,2,3" 문자열을 다시 쪼개고 숫자로 변환:



"1,2,3" → \["1","2","3"] → \[1,2,3]

"2,1" → \[2,1]

"1,2,4,3" → \[1,2,4,3]

"2" → \[2]



⭐ 최종 결과:

\[

&nbsp; \[1, 2, 3],

&nbsp; \[2, 1],

&nbsp; \[1, 2, 4, 3],

&nbsp; \[2]

]





이 Array.map() 메소드는 콜백 함수를 이용해 각각의 요소에 호출해서 그 값을 변환할 수 있게 해줍니다. 다시 말하자면 콜백 함수는 배열의 각 요소에 실행됩니다.
